# Curved Bottom Nav Bar Example

This is an example app demonstrating the usage of the `curved_bottom_nav_bar` package.

## Running the Example

1. Navigate to the example directory:
```bash
cd example
```

2. Install dependencies:
```bash
flutter pub get
```

3. Run the app:
```bash
flutter run
```

## Features Demonstrated

- ✅ Using Material Icons
- ✅ Multiple navigation items
- ✅ Custom colors per page
- ✅ Smooth animations
- ✅ Responsive design

## Customization

The example shows various ways to customize the navigation bar:

- **Using Material Icons**: Default implementation
- **Using Asset Images**: Commented code available
- **Color Customization**: Different colors per tab
- **Size Customization**: Comments show available options

Feel free to experiment with different configurations!
