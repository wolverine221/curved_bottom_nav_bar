# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-02-13

### Added
- Initial release of Curved Bottom Nav Bar
- Beautiful curved navigation bar with floating selected icon
- Smooth animation transitions
- Support for IconData, Image Assets, and Custom Widgets
- Fully customizable colors and styles
- Responsive design with flutter_screenutil
- Support for 2-5 navigation items
- Customizable curve depth and floating button size
- Shadow effects
- Gradient support for floating button
